## Features

- Increasing difficulty with each level  
- Dark mode user interface  
- Button flash animations  
- Game over feedback with screen shake  
- High score tracking (session-based)

---

## How to Play

1. Press any key to start the game  
2. Watch the sequence of flashing buttons  
3. Repeat the sequence by clicking the buttons  
4. Each correct round adds one more color  
5. A wrong click ends the game
